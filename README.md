# Morse3DS

A Morse code translator for Nintendo 3DS.

Morse3DS allows you to convert normal text into Morse code and convert Morse code back into text using the Nintendo 3DS controls.

## Controls

| Button | Function |
|---|---|
| A | Confirm / enter input |
| B | Switch between Text → Morse and Morse → Text modes |
| START | Exit the application |
